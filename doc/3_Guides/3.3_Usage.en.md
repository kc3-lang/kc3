# KC3 usage guide

## Running ikc3
```sh
ikc3/.libs/ikc3
```

## Running gdb
```sh
make gdb_ikc3
make gdb_test
make gdb_test_ikc3
```

## Running lldb
```sh
make lldb_ikc3
make lldb_test
make lldb_test_ikc3
```
