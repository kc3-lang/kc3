# 1.12 F128

The KC3 `F128` type is a C-compatible IEEE 754 floating point number
on 128 bits.

## 1.12.1 Examples

```elixir
ikc3> (F128) 1 - (F128) 0.001
(F128) 9.989999999999999999791833182882783e-1
```
