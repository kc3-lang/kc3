KC3 est un langage de programmation intégrant la métaprogrammation ainsi qu'une base de données orientée graphe directement dans le langage. Il vise à devenir le langage de référence pour la programmation sémantique et la programmation du web sémantique.

Une campagne de financement est en cours pour concrétiser le projet, et un prototype fonctionnel est disponible à l'adresse : https://git.kmx.io/kc3-lang/kc3/. Si vous souhaitez contribuer, veuillez consulter la page https://www.kmx.io/donations.
