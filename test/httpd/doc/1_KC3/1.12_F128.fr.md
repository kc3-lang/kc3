# 1.12 F128

Le type KC3 `F128` est un nombre flottant compatible C et IEEE 754
sur 128 bits.

## 1.12.1 Exemples

```elixir
ikc3> (F128) 1 - (F128) 0.001
(F128) 9.989999999999999999791833182882783e-1
```
