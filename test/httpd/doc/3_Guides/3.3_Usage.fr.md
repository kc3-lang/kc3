# Guide d'utilisation de KC3

## Lancer ikc3
```sh
ikc3/ikc3
```

## Utiliser gdb
```sh
make gdb_ikc3
make gdb_test
make gdb_test_ikc3
```

## Utiliser lldb
```sh
make lldb_ikc3
make lldb_test
make lldb_test_ikc3
```
